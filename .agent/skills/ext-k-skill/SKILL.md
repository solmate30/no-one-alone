# External K-Skill Library

This is a comprehensive collection of **Korean localized tools and services**. It provides specialized logic for interacting with various Korean platforms and data sources.

> **Source of Truth**: Refer to [README.md](./README.md) and the individual package directories for detailed documentation on each sub-skill.

## 1. Included Sub-Skills
- **Public Services**: Law search (`korean-law-search`), fine dust location (`fine-dust-location`).
- **Logistics & Transport**: Delivery tracking (`delivery-tracking`), KTX/SRT booking.
- **Ecommerce**: Coupang/Daiso product search.
- **Leisure**: KBO/K-League results, Lotto, neighborhood bar search.

## 2. Usage Instructions
1. Navigate to the specific subdirectory for the task at hand.
2. Read the local `README.md` or `SKILL.md` within that subdirectory (if available).
3. Follow the specific API or script instructions provided in the external library.
