# External Awesome Design Skill

This is an **externally maintained** design system library. Use it to draw inspiration for premium UI/UX aesthetics and to generate structured `DESIGN.md` files based on established visual identities.

> **Source of Truth**: Refer to [README.md](./README.md) and the [design-md/](./design-md/) directory for the full collection of design patterns and templates.

## 1. Capabilities
- **Aesthetic Reference**: Browse patterns from leading tech companies (Stripe, Vercel, Linear).
- **Template Generation**: Use the provided Markdown structures in `design-md/` to create consistent, high-quality documentation.
- **Visual Identity Audit**: Check project UI against the "wow" factor standards defined in these templates.

## 2. Usage Instructions
1. Browsing: Look into specific brand folders in `design-md/` to see their unique color palettes and layout rules.
2. Implementation: Copy and adapt Markdown snippets to your project's `docs/01_Concept_Design/` layer.
